# Retomar Trabalho — Início de Sessão

Execute no início de cada sessão de trabalho:

1. **Leia `docs/sprint-plan.md`** e identifique:
   - Qual fase está em andamento (primeira tarefa não marcada `[ ]`)
   - Quantas tarefas da fase atual já foram concluídas
2. **Leia `docs/session-log.md`** (última entrada) para saber:
   - O que foi feito na sessão anterior
   - Qual é a próxima tarefa a fazer
   - Se há problemas pendentes
3. **Resuma** em 3-5 linhas:
   - Onde estamos no projeto
   - O que foi feito até agora
   - O que vamos fazer agora
4. **Pergunte** se devo continuar com a próxima tarefa ou se há mudança de prioridade

Não comece a codar sem antes completar este resumo.
