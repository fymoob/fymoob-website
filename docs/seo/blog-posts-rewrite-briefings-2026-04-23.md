- Itaú: 11,60% | Bradesco: 11,70% | Santander: 11,69-11,79%

## 📋 Checklist de Publicação (todos os posts)

Pra garantir qualidade + honestidade + SEO:

- [ ] Título factualmente verificado (nenhuma afirmação sem fonte)
- [ ] Fonte citada no corpo (URL + data, não só link embutido)
- [ ] Data de publicação + "Atualizado em" no topo
- [ ] Lide em 2-3 frases (máx 280 chars) com dado concreto
- [ ] 5-8 H2s no formato pergunta ou afirmação (nunca rótulo)
- [ ] 3-5 curiosity loops entre seções
- [ ] Tabela ou callout visual nos primeiros 300 palavras
- [ ] 2 CTAs (meio + fim) sem "clique aqui" — ação específica
- [ ] 3-5 internal links (2 landing page, 1-2 blog pillar, 1 /contato)
- [ ] 1 imagem a cada 2 H2s, alt descritivo
- [ ] Reading time calculado
- [ ] FAQ schema se 3+ perguntas
- [ ] Schema Article + BreadcrumbList
- [ ] Meta description promete e entrega (sem clickbait vazio)
- [ ] **Disclaimer de data em dados financeiros:** "valores abril/2026"

## 🚨 Lição Crítica de Content Strategy

**6 dos 15 títulos originais eram factualmente incorretos.** Isso revela risco do padrão clickbait TNH1 quando aplicado sem validação:

- Google YMYL (finanças, moradia, saúde) pune desinformação severamente
- Portais grandes (TNH1, UOL) podem arriscar títulos polêmicos pq têm autoridade de domínio alta
- Site novo (fymoob.com.br, 6 dias pós-cutover) **NÃO tem** esse buffer

**Regra pra próximos títulos**: **toda afirmação no título precisa ser verificável no corpo com 2+ fontes**. Sem isso, não publica.

Esse processo de validação com agentes antes de reescrever **deve virar workflow padrão** pra todos os posts futuros.
